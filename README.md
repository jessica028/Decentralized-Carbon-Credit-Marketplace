# Decentralized Carbon Credit Marketplace
A transparent and efficient marketplace for trading carbon credits on the Bitcoin network via Stacks.

## 🎯 Features

- Create and verify carbon credits
- List credits for sale
- Purchase credits from the marketplace
- Track credit balances
- Transparent pricing mechanism

## 🚀 Getting Started

### Prerequisites

- Clarinet
- Stacks wallet

### Contract Functions

#### For Administrators
- `create-credits`: Mint new carbon credits
- `set-credit-price`: Update the base price of credits

#### For Users
- `list-credits`: List credits for sale
- `buy-credits`: Purchase credits from listings
- `get-balance`: Check credit balance
- `get-listing`: View specific listing details
- `get-credit-price`: Get current credit price

## 💡 Usage Example

1. Admin creates credits for verified carbon reduction
2. Users can list their credits for sale
3. Buyers can purchase credits using STX
4. All transactions are recorded on-chain

## 🔒 Security

- Owner-only functions for critical operations
- Balance verification for all transfers
- Automated price calculations
